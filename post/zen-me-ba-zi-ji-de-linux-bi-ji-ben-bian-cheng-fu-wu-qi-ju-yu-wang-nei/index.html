怎么把自己的linux笔记本变成服务器（局域网内）

2022-03-29 2 min read # ubuntu # linux

最近想一边在床上玩手机，一边控制自己的笔记本，所以就想把笔记本当作服务器，查了一下，果然可以

1.linux上安装openssh-server

sudo apt install openssh-sever

如果失败了，请安装aptitdue

sudo apt install aptitude

然后根据提示卸载冲突的依赖并安装openssh-server

2.手机上安装termux，
这里是下载链接 https://f-droid.org/en/packages/com.termux/
安装完成后。
打开termux，先根据提示更新软件

pkg upgrade

安装openssh

pkg install openssh

3.进行远程控制
ssh的使用方法是

ssh 用户名@ip地址

不会有人不知道自己的用户名把，打开终端@前面的就ifconfig是你的用户名，
当然，你也可以直接用root这个管理员名字，做用户名，好处是不用动不动就要输入sudo

ip地址，可以在ubuntu终端上输入

ifconfig | grep inet

inet 后面的是ipv4地址 （请不要输入127.0.0.1，那是本地地址）
inet6 后面的是ipv6地址（请不要输入::1，那是本地地址）

然后怎么做知道了吗？输入之后，按完确认建，会要求你输入yes，然后是你linux的密码，
登陆成功，撒花💐
